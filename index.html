Qui suis-je ?



Je suis Alexandre Turpin, étudiant en informatique à Polytech Tours. Passionné par l’informatique et plus particulièrement par la sécurité informatique, j’ai pour but de devenir expert dans ce domaine. De plus je suis également un grand fan de sport plus particulièrement de Crossfit en espérant pouvoir faire de belles choses dans ce sport.

Génération d'images artificielles grâce aux réseaux de neuronnes

Réalisation d'un programme capable de générer des images artificielles de silures à partir d’images aériennes de la Loire existantes. Pour cela nous avons utilisé les dernières architectures de Deep Learning (GAN).

PYTHON GAN TensorFlow Kerras

Logiciel de gestion d'un magasin de sport

Réalisation d'une application client-serveur Java et d'une base de données géré le système gestionnaire de base de données (SGBD) MySQL. Les utilisations de l’application sont la consultation et la gestion des stocks et des utilisateurs d’un magasin de sport.

JAVA JDBC JAVAFX MySQL Mockito Sonar JUnit

Logiciel de gestion d'annonces immobilières

Cette application permet à un utilisateur de s'enregistrer, d'ajouter une annonce, faire des offres et en accepter. De plus il est possibles d'effectuer des recherches parmis les annonces avec différents filtres.

JAVA JDBC SWING MySQL

Logiciel de tarification d'électricité

Ce logiciel permet à un utilisateur de rentrer la consomation électrique de ses compteurs afin d'obtenir le prix qu'il devra payer en fonction des heures pleines et creuses.

JAVA Hibernate JAVAFX MySQL

Librairie mathématique sur les matrices

Réalisation d'une librairie C++ permettant de faire des opérations sur des matrices. Opérations basiques (addition, multiplication...) et plus complexes (déterminant, inverse...), ainsi que la mise en place d'un parsing de fichier pour récupérer une matrice.

C++

Détection du stress à partir d'une vidéo

Ce projet avait pour but de développer un programme permettant de dire si une personne sur une vidéo est stréssé ou non. Pour ce faire j'ai utilisé la librairie OpenCV pour faire la détection du visage. Puis avec les paramètres obtenus par la détection et des outils de machine learning comme scikit-learn j'ai pu décider si la personne sur la vidéo etait stressé ou pas.

PYTHON OPENCV Scikit-learn
